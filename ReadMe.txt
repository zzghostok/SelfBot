step 1. find a vps of your choice (glitch.com is a free one)
step 2. open your terminal (on glitch.com find where it says tools then click full page console) and paste 
git clone https://github.com/Not-Cyrus/SelfBot

step 3. On glitch.com copy paste this command sequence: 
cd SelfBot
bash SetupLinux.sh

step 3.1. On any other LINUX vm just do ./ in replace for bash (you don't have to)
step 4. Get your token and put it in the file called Token.txt in the SelfBot folder 
step 5. go back to console and paste 
bash StartupLinux.sh
